Practical Unit Testing

Write great unit tests using the most popular technologies of the Java world!
Learn how to use JUnit, TestNG, Mockito and many other tools to unit test your code.
● Gain confidence that your code really works
● Develop code following the TDD approach
● Use mocks, stubs and test-spies intelligently
● Increase your awesomeness! ;)
