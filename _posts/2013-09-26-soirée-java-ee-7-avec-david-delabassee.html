--- layout: post title: Soirée Java EE 7 avec David Delabassee ---

Le Lorraine JUG (Java User Group de Lorraine) vous invite à sa prochaine soirée de conférence pour une présentation de Java EE 7, la plateforme dédiée au développement, déploiement et gestion d'applications multitiers en Java™.

17 octobre 2013, 18h30

TELECOM Nancy

193 avenue Paul Muller, 54602 Villers-lès-Nancy (plan d'accès)

Au programme de la soirée :

18:15 - Accueil et enregistrement

18:30 - Java EE 7, David Delabassee (Principal Product Manager - GlassFish & Java EE, Oracle)

Entre JavaOne et Devoxx, David nous fait l'amitié de passer par le Lorraine JUG pour nous parler Java EE 7 et Glassfish. Il aura carte blanche pour nous faire découvrir les nouveautés de cette plateforme qui ne cesse d'évoluer depuis 1999.

21:30 - Drink, tombola…
